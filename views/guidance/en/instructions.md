# Instructions

## Overview

| Instruction | Description                                                                                                   |
|-------------|---------------------------------------------------------------------------------------------------------------|
| `add`       | Add                                                                                                           |
| `addi`      | Add (Immediate)                                                                                               |
| `beq`       | Branch If Equal |
| `beqz`      | Branch If Equal to Zero |
| `bnez`      | Branch If Not Equal to Zero |
| `blez`      | Branch If Less Than or Equal to Zero |
| `bgez`      | Branch If Greater Than or Equal to Zero |
| `bltz`      | Branch If Less Than Zero |
| `bgtz`      | Branch If Greater Than Zero |
| `bgt`       | Branch If Greater Than |
| `ble`       | Branch If Less Than or Equal |
| `bgtu`      | Branch If Greater Than (Unsigned) |
| `bleu`      | Branch If Less Than or Equal (Unsigned) |
| `ecall`     | Environment Call |
| `j`         | Jump |
| `jal`   | Jump and Link |
| `jalr` | Jump and Link with Register |
| `jr` | Jump to Register |
| `la`        | Load Address: Loads the address into a register\. e\.g\. \`la $a0,address`                                    |
| `lb` | Load Byte |
| `lbu` | Load Byte (Unsigned) |
| `ld` | Load Double-word |
| `lh` | Load Half-word |
| `lhu` | Load Half-word (Unsigned) |
| `li`        | Load Immediate: The li pseudo instruction loads an immediate value into a register\. e\.g\. \`li $8, 0x3BF20` |
| `lw` | Load Word |
| `lwu` | Load Word (Unsigned) |
| `mul` | Multiply |
| `mulh` | Multiply and return High |
| `mulhsu` | Multiply and return High (Signed x Unsigned) |
| `mulhu` | Multiply and return High (Unsigned) |
| `mulw` | Multiply Word |
| `mv` | Move |
| `nop` | No Operation |
| `not` | Not |
| `neg` | Negate |
| `negw` | Negate Word |
| `seqz` | Set If Equal to Zero |
| `snez` | Set If Not Equal to Zero |
| `sltz` | Set If Less Than Zero |
| `sgtz` | Set If Greater Than Zero |
| `or` | Logical Or |
| `ori` | Logical Or (Immediate) |
| `rem` | Remander |
| `remu` | Remander (Unsigned) |
| `remw` | Remainder Word |
| `remuw` | Remainder Word (Unsigned) |
| `sb` | Store Byte |
| `sd` | Store Double-word |
| `sh` | Store Half-word |
| `sll` | Shift Left Logical |
| `slli` | Shift Left Logical (Immediate) |
| `sllw` | Shift Left Logical Word |
| `slt` | Set Less Than |
| `slti` | Set Less Than (Immediate) |
| `sltu` | Set Less Than (Unsigned) |
| `sltiu` | Set Less Than (Immediate) (Unsigned) |
| `sra` | Shift Right Arithmetic |
| `srai` | Shift Right Arithmetic (Immediate) |
| `sraw` | Shift Right Arithmetic Word |
| `sraiw` | Shift Right Arithmetic Word (Immediate) |
| `srl` | Shift Right Logical |
| `srli` | Shift Right Logical (Immediate) |
| `srlw` |  Shift Right Logical Word |
| `srliw` | Shift Right Logical Word (Immediate) |
| `sub` | Subtract |
| `subw` | Subtract Word |
| `sw` | Store Word |
| `xor` | Logical XOR |
| `xori` | Logical XOR (Immediate) |
